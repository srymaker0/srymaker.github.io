线程栈

  • 定义的局部变量为被压入系统栈中
void funcA() {
    int a, b, c;
    funcB();
    return ;
}

void funcB() {
    int d;
}
  • 如上，局部变量a，b，c，d会依次被压入系统栈中。

  • 我的机器行默认栈大小为8192KB，也就是8M

  • 爆栈

    • 比如我们在调用递归函数的时候，递归深度过深，数据大小超过了8M，这么栈存不下数据，就会引起爆栈
    • 一个int类型占4字节，8M约800万字节，也就是存200万个int类型变量也会爆栈

表达式求值

  • 理解
    • 给你一个表达式，根据表达式求出最终的值
    • 右边的被称为表达式树——以运算符作为根节点，操作数作为相应的子节点

  • 对于这样一个表达式怎么算$3 \times(3 + 5)$

    • 我们可以设置优先级，根据优先级判断，”-+” 设置1优先级，”*/“设置2优先级
    • 对于括号内的运算符，计算其优先级时加100
      • 比如$3 \times(3 \times (3 + 5))$
      • 第一个✖️号优先级为2， 第二个✖️优先级为102（2 + 100），第三个➕号优先级为201（1 + 100 + 100 因为有两层括号）

  • 可以用递归，也可以用栈

    • 两者本质上并无区别

面试题 03.04. 化栈为队

class MyQueue {
public:
    /** Initialize your data structure here. */
    stack<int> s1, s2;
    MyQueue() {

    }
    
    /** Push element x to the back of queue. */
    void push(int x) {
        s1.push(x);
        return ;
    }
    
    /** Removes the element from in front of queue and returns that element. */
    int pop() {
        if (s2.empty() == 1) {
            while (!s1.empty()) {
                s2.push(s1.top());
                s1.pop();
            }
        }
        int ret = s2.top();
        s2.pop();
        return ret;     
    }
    
    /** Get the front element. */
    int peek() {
        if (s2.empty() == 1) {
            while (!s1.empty()) {
                s2.push(s1.top());
                s1.pop();
            }
        } 
        return s2.top();    
    }
    
    /** Returns whether the queue is empty. */
    bool empty() {
        return s1.empty() && s2.empty();
    }
};

1249. 移除无效的括号

  • 思路
    • 遍历两边，先遍历原字符串，删除多余的’)’括号，得到s1
    • 接着遍历s1，删除多余的’(‘括号即可
  • 代码
class Solution {
public:
    string minRemoveToMakeValid(string s) {
        string ret1;
        string ret2;
        int cnt1 = 0;
        for (int i = 0; i < s.size(); i++) {
            if (s[i] != '(' && s[i] != ')') {
                ret1 += s[i];
            } else {
                if (s[i] == '(')  {
                    cnt1++;        
                    ret1 += s[i];
                } else {
                    if (cnt1 > 0) {
                        cnt1--;       
                        ret1 += s[i];              
                    }  
                }
            }
        }
        int cnt2 = 0;
        for (int i = ret1.size() - 1; i >= 0; i--) {
            if (ret1[i] != '(' && ret1[i] != ')') {
                ret2 += ret1[i];
            } else {
                if (ret1[i] == ')')  {
                    cnt2++;        
                    ret2 += ret1[i];
                } else {
                    if (cnt2 > 0) {
                        cnt2--;       
                        ret2 += ret1[i];                
                    }
                }
            }
        }
        reverse(ret2.begin(), ret2.end());
        return ret2;
    }
};

331. 验证二叉树的前序序列化

  • 解题方法——缩点思维
    • 若数字后面有两个“#”，那么把这个数字和后面两个“#”合并成一个“#”，例如示例1👇
    • 第一次合并后变为“93#1##2#6##”，第二次“93##2#6##”，第三次“9#2#6##”，第四次“9#2##”，第五次“9##”，第六次“#”
    • 当且仅当只剩下一个“#”后我们认为这个序列是合法的前序序列

  • 注意要处理“，”

  • 将字符存入字符串数组中
